# slipmint-portal

A small portal demo showing how to connect wallets and mint/test tokens (testnet only).

Quickstart (demo)
1. Open `demo/index.html` in a browser (or host as a static site).
2. Switch MetaMask to BSC Testnet or Polygon Mumbai.
3. Click "Connect Wallet" to view your address.

Files
- `demo/` — runnable front-end demo (index.html + JS)
- `contracts/` — optional solidity examples
- `CONTRIBUTING.md` — how to contribute and claim bounties
